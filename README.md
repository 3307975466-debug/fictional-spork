**本项目是浙江树人学院《计算机图形学》期中考核作业**
## 项目简介

本项目是一个基于 Three.js 开发的数字孪生校园交互系统，可在浏览器中运行。系统采用 HTML + JavaScript + Three.js 技术栈，构建了一个具有基本空间结构和交互功能的三维校园场景，包含建筑、道路、植被等对象，支持鼠标浏览、对象点击信息展示、视角切换和简单动画等功能。

### 核心功能
1. 三维场景展示
   - 使用 Three.js 构建逼真的三维校园场景
   - 包含教学楼、图书馆、学生公寓等建筑
   - 道路系统、绿化植被、休闲设施等

2. 视角切换
   - 提供透视视角和鸟瞰视角两种模式
   - 支持鼠标缩放、旋转、平移控制
   - 视角平滑过渡动画

3. 信息展示
   - 点击建筑显示详细信息
   - 弹出信息面板展示建筑名称、功能、楼层等
   - 动态高亮显示选中对象

4. 图层控制
   - 可切换显示/隐藏不同类型的对象
   - 建筑图层、道路图层、植被图层独立控制

5. 动画效果
   - 旋转动画展示

### 框架技术
- Three.js: 三维渲染引擎
- HTML5/CSS3: 页面结构与样式
- JavaScript ES6+: 交互逻辑
- GLTF/GLB: 3D模型格式
